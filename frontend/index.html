About This Platform

A modular preprocessing system for single-cell RNA sequencing data standardisation.

🧬

Gene Identifier Normalisation

Converts diverse gene symbols to the GENCODE M32 reference using a curated local mapping table. Achieves >95% mapping success rate without network dependency.

🔬

Cell Type Ontology Alignment

Two-stage local algorithm: exact match against a 6,120+ synonym Cell Ontology table, followed by fuzzy matching fallback. Achieves 97.4% accuracy on a 538-label benchmark.

📊

Incremental Update SOP

Semi-automated pipeline for expanding the cell type lookup table as new datasets are processed, with automated before/after evaluation reporting.

📋

Reproducible Reporting

Every transformation is logged in a machine-readable JSON audit report, enabling full traceability of all mapping decisions.
