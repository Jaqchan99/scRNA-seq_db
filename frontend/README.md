# scRNA-seq 数据处理平台 - 前端

## 功能概述

这是一个简单而功能完整的前端界面，与后端API完全对齐，实现了scRNA-seq数据处理的完整流程。

## 主要功能

### 1. 数据上传
- 支持拖拽上传 .h5ad 文件
- 文件格式验证
- 文件大小显示
- 上传进度提示

### 2. 数据校验（逐步确认）
- 实时显示校验进度
- 展示校验结果摘要
- 显示错误和警告信息
- 数据质量指标预览
- **用户确认后才能继续下一步**

### 3. 基因映射与细胞类型标准化（逐步确认）
- 显示映射进度
- 展示映射成功率
- 显示映射详情表格
- 支持查看前10条映射结果
- **用户确认后才能继续下一步**

### 4. 结果展示
- 处理完成确认
- 数据统计摘要
- 质量指标展示
- 结果文件下载

## 处理流程

采用**逐步确认模式**，每个步骤完成后需要用户确认才能继续：

1. **上传文件** → 自动进入校验步骤
2. **数据校验** → 展示校验结果 → **用户确认** → 进入映射步骤
3. **基因映射与细胞类型标准化** → 展示映射结果 → **用户确认** → 进入导出步骤
4. **结果展示** → 下载处理结果

## 使用方法

### 方法1: 使用启动脚本（推荐）
```bash
# 在项目根目录运行
python start_platform.py
```

### 方法2: 手动启动
```bash
# 1. 启动后端API服务
python main.py

# 2. 在另一个终端启动前端服务
cd frontend
python -m http.server 6666
```

### 方法3: 直接打开文件
```bash
# 确保后端服务已启动
python main.py

# 直接打开 frontend/index.html 文件
```

## 界面说明

### 步骤指示器
- 显示当前处理步骤
- 已完成步骤显示为绿色
- 当前步骤显示为蓝色

### 上传区域
- 支持拖拽和点击上传
- 仅接受 .h5ad 格式文件
- 显示文件基本信息

### 结果展示
- 卡片式布局展示关键指标
- 表格形式展示详细结果
- 支持数据下载

## 技术特点

- **纯前端实现**: HTML + CSS + JavaScript
- **响应式设计**: 支持桌面和移动设备
- **实时更新**: 轮询API获取处理状态
- **错误处理**: 友好的错误提示和重试机制
- **无依赖**: 不需要额外的框架或库

## 与后端API的对应关系

| 前端功能 | 后端API | 说明 |
|---------|---------|------|
| 创建提交 | POST /submissions | 创建新的处理任务 |
| 文件上传 | POST /submissions/{id}/upload | 上传h5ad文件 |
| 状态查询 | GET /submissions/{id}/status | 查询处理状态 |
| 获取报告 | GET /submissions/{id}/report | 下载处理报告 |
| 下载数据 | GET /submissions/{id}/export | 下载处理结果 |

## 注意事项

1. 确保后端API服务在 http://localhost:8100 运行
2. 仅支持 .h5ad 格式文件
3. 大文件上传可能需要较长时间
4. 处理过程中请勿关闭浏览器

## 故障排除

### 无法连接后端
- 检查后端服务是否启动
- 确认端口8100未被占用
- 检查防火墙设置

### 文件上传失败
- 确认文件格式为 .h5ad
- 检查文件大小是否超限
- 查看浏览器控制台错误信息

### 处理卡住
- 检查网络连接
- 查看后端服务日志
- 尝试重新开始流程
